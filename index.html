What is Event Bubbling?

Event Bubbling is when click on the third div that you have nested 3 div boxes before, it triggered from bottom to top like from box-3 to box-2 and box-1. (Opens from the inside-out.)

What is Event Capturing?

Event Capturing is Although when you click on the third div that you have nested 3 div boxes before, it triggered from top to bottom like from box-1 to box-2 and box-3. (Opens from the outside-in.)

As default, JavaScript set the event flow as Bubble.

If we want to use Capturing, we should use the third argument in addEventListener() as true.

So, What is Event Delegation?

When we work with bigger data table like it has 10 grids and 100 rows. If we want to make every cells can be editable when they click.
We need to add addEventListener() to all cells and this will be a huge performance problem that cause browser crash.

Instead of this, we can add only one event listener to table of parent using with Event Delegation, we can stop the click event and then we can determine to which cell has been clicked!
